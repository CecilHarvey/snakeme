Introduction :

Les regles de base sont tres simples: Il s'agit d'un serpent qui doit manger des pommes pour grandir et gagner des points. Ce genre de jeu existe deja dans sur telephones portables ainsi que dans de nombreux sharwares et freewares

Qu'y-a-t'il donc d'unique et de nouveaux dans SnakeMe?

D'abord, c'est un tres joli jeu avec de belles couleurs, et une interface ergonomique.

On peut y jouer seul, mais aussi avec un maximum de trois autres personnes. Et cela ne necessite pas d'avoir un ordinateur par joueur: deux personnes peuvent jouer sur le meme clavier, et les deux autres avec des joypads,ou joysticks si vous en avez. Il peut y avoir jusqu'a huit joueurs par jeu, donc , si il y a, par exemple deux personnes qui jouent, il est possible d'ajouter 6 serpents controles par l'ordinateur (AI).

Il y a aussi une arme, mais attention: les munitions ne sont pas infinies et il faut les ramasser.

SnakeMe est un jeu totalement configurable: vous pouvez choisir vos niveaux, vos serpents, et les couleurs que vous voulez voir sur le niveaux que vous aurez choisi. Il est aussi possible de creer ses propres niveaux grace a un editeur tres facile a utiliser. c'est egalement possible de creer d'autres serpents ou d'autres themes grace a un petit logiciel fourni avec SnakeMe : spriteEditor. Et vous pouvez egalement telecharger d'autres serpents, niveaux, et themes depuis notre site.

Configuration Minimale Requise :

  • Systeme d'exploitation :
    Windows (95, 98, Millennium, NT4, 2000)
    Linux (x86)
    Macintosh (PPC)
    Il y aura peut etre ulterieurement d'autres systemes supportes..
  • Processeur :
    Pour Windows ou Linux: un ordinateur de generation 486 dx2 (ou mieux).
    Pour Macintosh: un ordinateur de generation powerPC (ou mieux).
  • Carte Graphique :
    Une carte graphique SVGA 640x480x16 bpp (fonctionne aussi avec une 8 bpp mais moins joliment, et avec une 32 bpp mais plus lentement).
  • Carte Son :
    Une carte son soundBlaster 16 ou compatible (44'100Hz 16bit stereo - fonctionne aussi avec moins mais le son n'est pas excellent).
  • RAM :
    8 Mo de RAM (recommande: 16 Mo ).
  • Disque Dur :
    4 Mo d'espace libre sur le disque dur.
  • Periferiques :
    Un clavier et une souris.
    Les joysticks et joypads sont supportes.

Installation sous Windows :

  • Double-cliquez sur l'icone "snakeme.win32.exe".
  • Indiquez l'emplacement ou vous voulez installer SnakeMe, ou cliquez sur cet emplacement apres avoir choisi le bouton "browse" .
  • Voila, amusez-vous bien!

Lancer SnakeMe sous Windows :

  • Cliquez dans le menu demarrer pour aller sous Programmes, SnakeMe.
  • Choisissez quelle version (anglaise, francaise) vous voulez lancer en cliquant sur le racourci approprie.

Si vous rencontrez des problemes dans le lancement de SnakeMe, allez la ou vous l'avez installe et double-cliquez sur l'icone "SnakeMe.exe".

Desinstallation sous Windows :

  • Cliquez dans le menu demarrer pour aller sous Programmes, SnakeMe.
  • Cliquez sur le racourcis "Uninstall".
  • Suivez les instructions.

L'installation est de toutes facons propre et touts les fichiers sont dans le meme repertoire. C'est pourquoi vous pouvez aussi desinstaller manuellement en effacant simplement le repertoire SnakeMe, et aucun fichier ne restera.

Installation sous Macintosh :

  • Cliquez sur l'icone "snakeme.macos".
  • Suivez les instructions.
  • Installez ensuite l'extention SDL.

Lancer SnakeMe sous Macintosh :

Allez dans le dossier ou vous avez installe SnakeMe et lancez-le.

Desinstallation sous Macintosh :

Effacez le dossier SnakeMe et l'extention SDL.

Linux :

Prenez snakeme.tar.gz, et placez le votre repertoire favori.

Decompressez le fichier avec la commande : tar xvzf snakeme.tar.gz

Puis allez dans le repertoire cree en tappant : cd snakeme

Et lancez finalement SnakeMe avec : ./snakeme

Menu Principal :

Vous voyez cinq boutons :

  • Jeu :
    Vous permet de jouer a SnakeMe.
  • Controles :
    Vous permet de personnaliser les touches avec lesquelles vous voulez jouer.
  • Options Avancees :
    Vous permet de personnaliser des parametres et des regles du jeu.
  • Editeur de Niveau :
    Vous permet de dessiner vos propres niveaux.
  • Quitter :
    Vous permet de fermer SnakeMe.

Configuration du Jeu :

Cet ecran est partage en trois zones differentes :

En haut, d'abord vous pouvez choisir votre niveau (a gauche) et votre theme (couleurs) (a droite). utilisez les fleches sous l'image pour changer la selection. Vous pouvez jouer au niveau de votre choix avec n'importe quel theme.

Au milieu, ensuite, il y a huit boites de configuration pour huit joueurs. Chaque boite permet de :

  • Activer ou non le joueur grace au bouton vert ou rouge, vert pour un joueur active, rouge pour un joueur desactive.
  • Indiquer le nom du joueur grace au champs de texte.
  • Choisir le mode de controle du serpent en cliquant sur l'image representant des fleches. Quatre fleches representent le mode absolu, c'est a dire relatif a l'ecran. Deux fleches representent le mode relatif, c'est a dire relatif au serpent.
  • Choisir qui joue et comment en cliquant sur l'image qui se transforme en A, en B, en AI, en A joy ou en B joy. En A vous utilisez les touches de controle A. En B vous utilisez les touches de controle B. Vous pouvez personnaliser ces touches dans la section controles (en cliquant sur controles au bas de l'ecran). En AI l'ordinateur controle le serpent. Si vous avez des joysticks ou des joypads, vous pouvez les utiliser en choisissant A joy ou B joy.
  • Choisir avec quel serpent vous voulez jouer en cliquant sur les fleches qui se trouvent sous l'image du serpent..

En bas, enfin, vous pouvez soit retourner au Menu Principal (bouton Anuller), commencer a jouer (bouton Jouer), personnaliser les touches de controle (bouton Controles), ou personnaliser les parametres et regles du jeu (bouton Configuration Avancee) (voir plus bas).

Jouer :

Vous pouvez voir en haut les differents serpents qui jouent, le nom du joueur qui controle chacun et ses points . Si un serpent a ramasse des munitions une petite barre bleu apparait sous le nom du joueur. Plus la barre est longue plus il a de munitions.

Lorsque quelqu'un atteint le nombre de points necessaires pour gagner, le jeu est fini et un autre ecran apparait pour indiquer le score de chacun. Si vous voulez arreter avant que quelqu'un n'ait gagne, appuyez sur la touche ESC.

La maniere de compter les points est definie dans la section de Options Avancees. Pour gagner des points on peut ramasser de la nouriture, toucher un autre serpent avec une munition, ou faire en sorte qu'un autre serpent se cogne contre le sien. On perd des points lorsqu'on est touche par une munition et lorsqu'on se cogne contre quelque chose.

Amusez-vous bien!

Controles :

Ici, vous pouvez personnaliser les touches avec lesquelles vous jouerez. Cliquez sur le bouton representant l'ancienne touche et tapez la nouvelle.

Il y a deux possibilites differentes pour controler votre serpent: avec quatre touches (mode absolu), ou deux (mode relatif). si vous etes en mode absolu et que vous appuyez, par exemple, sur la touche "droite-absolu", votre serpent ira a droite.Mais en mode relatif, si vous appuyez sur la touche "gauche-relatif", votre serpent tournera de 90 degres dans le sens inverse des aiguilles d'une montre. Vous pouvez choisir si vous voulez jouer en mode relatif ou absolu dans l'ecran "Configuration du jeu".

La touche action sert a envoyer les munitions que vous aurez ramassee pendant le jeu.

Options Avancees :

Ici, vous pouvez personnaliser les parametres et regles du jeu. Vous pouvez ainsi decider du deroulement du jeu , et vous amusez encore plus. Il suffit de bouger la barre pour ajuster l'option desiree.

Voici une breve description de chaque option :

  • Vitesse du jeu :
    Permet de choisir le nombre d'images par seconde (fps), c'est a dire la vitesse a laquelle se deplacent les serpents.
  • Longueur du serpent par defaut :
    Permet de choisir la longueur du serpent au moment ou il apparait.
  • Longueur d'allongement du serpent :
    Permet de choisir de combien le serpent grandit chaque fois qu'il ramasse de la nouriture.
  • Nombre de nouriture :
    Permet de choisir la quantite constante de nouriture qu'il y a dans le jeu.
  • Temps de changement de la nouriture :
    La nou riture change regulierement de place. Cette option permet de choisir le temps entre les changements de place.
  • Nombre de bonus de tir :
    Permet de choisir la quantite constante de munition qu'il y a dans le jeu.
  • Temps de changement des bonus :
    Les munitions changent regulierement de place. Cette option permet de choisir le temps entre les changement de place.
  • Nombre de point pour gagner :
    Permet de choisir le nombre de points a atteindre en premier pour gagner.
  • Point gagnes quand on mange :
    Permet de choisir le nombre de points que l'on gagne lorsqu'on ramasse de la nouriture.
  • Points gagnes quand on tire:
    Permet de choisir le nombre de points que l'on gagne lorsqu'on touche un autre serpent avec une munition..
  • Points gagnes quand on bloque :
    Permet de choisir le nombre de points que l'on gagne lorsqu'un autre serpent se cogne contre le sien.
  • Points perdus quand on s'ecrase :
    Permet de choisir le nombre de points que l'on perd lorsque son serpent meurt.
  • Points perdus quand on est touche :
    Permet de changer le nombre de points que l'on perd lorsqu'on est touche par une munition.

Editeur de Niveau :

Ici vous pouvez dessiner vos propres niveaux. C'est un simple programme de dessins

La premiere chose a savoir est qu,il y a un petit texte d'aide qui apparait en haut a droite si vous mettez votre souris sur un bouton.

Il y a six boutons en haut a gauche :

  • Sortir
    Permet de fermer l'Editeur de niveau et de retourner au jeu. Si vous n'avez pas sauvegarde votre niveau, une boite de dialogue apparait pour vous demander si vous voulez le faire ou non.
  • Charger
    Ouvre un niveau existant, que vous pouvez ainsi modifier.
  • Nouveau
    Cree un nouveau niveau.
  • Sauver
    Vous permet de sauvegarder votre niveau. Si vous entrer le nom d'un niveau existant (ou si vous cliquez sur un niveaux existant), votre niveau remplacera celui existant sans avertissement.
  • Theme
    Vous permet de voir a quoi votre niveau ressemble avec le theme de votre choix. Mais attention, le theme choisit ne sera pas attribue a votre niveau: il est toujours possible de jouer a n'importe quel niveau avec n'importe quel theme.
  • Annuler
    Vous permet d'annuler une action (mais seulement une).

En haut au milieu, il y a quatre outils de dessins: le premier dessine des points, le second des lignes, le troisieme des rectangles, et le dernier remplit la zone donnee.

Et en haut a droite vous pouvez choisir se que vous dessiner (la couleur de fond, un mur incassable, ou un mur cassable).
